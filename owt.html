Why oldweb.today?

  • Difficult problems in web archive replay remain
  • A moving target as the web evolves
  • Effectivity of emulation approach: a dozen containers for millions of web pages

Web Archive Replay Problems

and Solutions!

  • Web Archive 'leaks' to live web

    • Live leaks not possible (Proxy Mode)

  • Improperly rewritten dynamic urls

    • No Rewriting needed! (Proxy mode)

  • Pages just don't "look the same" (no <blink>)

    • Use era-appropriate browsers (Emulation)

  • Flash, Java, Shockwave, other legacy content

    • No problem! (Proxy Mode + Emulation)

Architecture and Concepts

A layered Architecture

  • Web archives provide web pages (Memento)
  • Web pages running in Browsers (Emulation)
  • Browsers running in Emulators (Emulation)
  • Emulators running in Containers (Emulation)
  • Containers running in a VM (Amazon EC2)

Client-Side

  • Containers stream screencast to user's browser
  • Using VNC Protocol over WebSockets to draw on HTML5 Canvas
  • All handled nicely by the noVNC JS library
  • Still missing: audio support

Emulators: What is being emulated?

Old Browsers and Operating Systems

  • The browser is the platform for web pages
  • The OS is secondary to the browser
  • Differences from other software emulators:
    • CPU, OS precision less important
    • Latency OK
    • New challenges: networked emulators

Web (HTTP) emulation

  • Not general Internet emulation
  • HTTP is our friend here!
  • HTTP Protocol has not changed much over 20 years!
  • HTTP proxy mode support makes emulation + web archiving possible.

HTTP Proxy

  • Easy mechanism for serving archived web content instead of original.
  • Prohibitive to configure manually just to view an archive...
  • ...but easy if pre-configured in a "container"

Containers

  • More lightweight than VMs, use less resources
  • 'Contain' programs, data and file systems that can be deployed rapidly
  • Perfect for launching multiple instances of complex systems on same machine (VM)
  • oldweb.today supports roughly 5-6 browser containers per EC2 CPU

Docker

  • Popular open-source container system
  • Manages containers, providing:
    • Scheduling, Resource sharing and limits
    • Networking
    • Security
    • Portable Container Image Format
    • Lots of APIs
    • Large, open source community!

  • No need to "reinvent wheel"

Memento Aggregation

  • Uses ODU Memento Aggregator by Sawood Alam
  • Aggregates from multiple TimeGate API calls
  • Finds closest match from available archives
  • (Almost any) Memento-compatible archive can be included...
  • Almost.. must serve unaltered archived web content

Cultural Impact

The web as performative media

oldweb.today makes the variability of the web tangible.

The web has, since the advent of client-side scripting in 1995, been a software delivery system rather than a document delivery system.

Software Curation

Absence of technically technically complex artifacts in classic web archives

As web archiving techniques improve (for instance with Webrecorder), the preservation of software environments that the web can be performed within becomes essential.
